# Uploade de Arquivos

Upload de arquivos usando NodeJS e ReactJS, amobos utilizando typescript.

## Backend

Bibliotecas usadas

- Typescript
- Express
- Multer
- DotEnv
- Cors

#### Instalação

Criar arquivo `.env` e settar a propriedade `APP_PORT`

```
cd server
yarn
yarn dev
```

ou

```
npm install
npm dev
```

## FrontEnd

Bibliotecas usadas

- React
- Typescript
- Axios
- Formik
- Yup

#### Instalação

Criar arquivo `./src/services/api.tsx` e settar a propriedade `baseURL`

```
yarn
yarn start
```

ou

```
npm install
npm start
```
